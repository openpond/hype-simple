# Hyperliquid 200 SMA

[![Deploy on OpenPond](https://img.shields.io/badge/Deploy_on-OpenPond-1E90FF?style=flat)](https://openpond.ai/project/new?repo=openpond/hyperliquid-200-sma&ref=readme-host-button)

A simple strategy that places a long order on the 1minute 200 SMA every 10 minutes.

## Run locally

```bash
bun install
bunx opentool dev

set env variables in .env.example file or host on OpenPond
```
